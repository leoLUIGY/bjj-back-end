# 🥋 BJJ Catalog API

API para gerenciamento de times de Jiu-Jitsu, professores e informações relacionadas ao esporte.

---

## 🚀 Tecnologias utilizadas

- Python 3.11+
- Flask
- Flask-OpenAPI3 (Swagger)
- SQLAlchemy
- SQLite
- Flask-CORS
- Pydantic

---

## 📦 Instalação

### 1. Clonar o repositório
```bash
git clone https://github.com/seu-usuario/bjj-back-end.git
cd bjj-back-end

### 2. Instalar dependencias
```bash
pip install -r dependencies.txt

### 3. Executar api
```bash
python app.py

Abra o http://127.0.0.1:5000/openapi/swagger no navegador para verificar o status da API em execução.
